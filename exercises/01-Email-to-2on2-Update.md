# Exercise 1 — Inbox Summary & Draft (Outlook)

- **App:** Outlook on the web
- **Workflow:** Turn long CCIIO threads into briefings and action items, then draft the replies and notes you'd otherwise write from scratch — in your voice
- **Material:** the four practice threads from the [Practice Inbox Pack](../assets/Practice-Inbox-Pack.md)
- **Estimated time:** 18 minutes
- **Difficulty:** Core

---

## Objective

CCIIO work runs on email — data calls, contract mods, compliance questions, leadership asks. This exercise uses Copilot in Outlook to **summarize** those threads, **pull the action items**, and **draft** the replies and notes, always leaving the send button to you.

## Before you start

- [ ] You completed the pre-work and emailed yourself the four practice threads.
- [ ] Outlook on the web is open and you can see the four threads by subject.

## What Copilot is doing here

Copilot in Outlook works over **the mail you can open**. To point it at a thread, type a forward slash `/` in the Copilot prompt, start typing the subject, and select it from the list — then run your prompt. Copilot **sends nothing**; every draft waits for you to review and click send.

---

## Part A — Catch up & brief

### Step 1 — Catch me up
In Copilot, run:
```text
Catch me up on my email from the last few days. Group by topic; for each give a one-line status, any deadline, and who owns the next step. Flag anything that needs a decision from me.
```

### Step 2 — Thread → 2-on-2 status update
In Copilot, type `/`, start typing **DMID Data Call — Response Status**, and select it. Then run:
```text
Summarize this thread into 5 bullets for my 2-on-2 with my manager: key decisions, open action items with owners, current status, any risks or blockers, and next steps. Professional status-update tone. End with one line: "Decision needed from you."
```

### Step 3 — Action items only
In Copilot, type `/`, start typing **Contract Mod #14 — Northwind Extension**, and select it. Then run:
```text
From this thread, list only the open action items as a checklist: owner, what they committed to, and the due date. Leave out anything already resolved.
```

---

## Part B — Draft as a professional

### Step 4 — Reply in your voice
In Copilot, type `/`, start typing **Compliance Audit Scope — Vendor Gap**, and select it. Then run:
```text
Draft a reply from me confirming I'll set up the 10:00 scope call, bring the current scope doc with the vendor-assessment gap marked for a live redline, and target the revised scope back to Compliance by Friday. Professional, three short paragraphs. Don't send — I'll review.
```
Read it aloud. If it doesn't sound like you, tell Copilot what to change (shorter, warmer, more direct) — don't accept a draft that isn't your voice.
