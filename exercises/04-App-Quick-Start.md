# App Quick-Start — Copilot Across Your M365 Apps (Session 2A)

- **Apps:** Outlook, Teams, Word, Excel
- **Time:** ~10 minutes (2–3 minutes each)
- **What you'll do:** A fast tour — one quick win in each app, using the *demo → replicate → refine* rhythm from the session.

---

## How to run this
For each app: the facilitator demos it, you do the same on your own (non-sensitive) content, then you refine with one follow-up. Each app teaches one prompt move worth keeping.

## Outlook — *audience framing*
```
Summarize this email for the top 3 main points and any action items I need to take.
```
Then reframe for a different reader: *"Now summarize it for my manager who only needs the decision and the risk."* Notice how the output changes with the audience.

## Teams — *turn a call into documentation*
```
Recap this meeting and list my action items with owners and due dates.
```
Then: *"Draft a short follow-up message to the team with the decisions and next steps."*

## Word — *constraints sharpen output*
```
Rewrite this in plain language, under 150 words, with no acronyms: [paste your text]
```
Then: *"Make it warmer and more conversational."* Constraints (length, tone, no-jargon) make the result more usable than an open-ended ask.

## Excel — *ask for a question, not just an answer*
```
Explain what this data shows in plain English and flag anything unusual.
```
Then: *"What would a skeptical reviewer ask about this data?"* Asking for the *question* often surfaces better insight than asking for a summary.

## Remember
- **Verify** names, dates, and numbers against the source before you use anything.
- **Iterate** — your first prompt is a starting point, not the finish line.
- **Never** paste sensitive or non-public information.
